# Weather App - Real-time Weather Information Platform

A responsive web application that delivers real-time weather information for cities worldwide. Users can search any city and instantly receive comprehensive weather data including temperature, humidity, wind conditions, and interactive location mapping.

---

## Project Overview
**Project Type:** Frontend Web Application

People need quick, accurate weather information without navigating complex interfaces or dealing with cluttered designs. This application provides instant weather data with an intuitive, clean interface that works seamlessly across all devices.

---

## Tech Stack

### Frontend Technologies
- **HTML5** - Semantic markup and structure
- **CSS3** - Custom styling with gradients, animations, and responsive design
- **JavaScript ES6+** - Object-oriented programming with modern syntax
- **Bootstrap 5.3** - Responsive grid system and UI components
- **Font Awesome 6.4** - Professional icon library

### APIs and Libraries
- **OpenWeatherMap API** - Real-time weather data integration
- **Leaflet.js 1.9.4** - Interactive mapping and geolocation visualization

---

## Key Features

- **Real-time Weather Data** - Current temperature, humidity, wind speed, pressure, and visibility for any location
- **Global City Coverage** - Search weather for over 1000 cities worldwide with comprehensive Indian city database
- **Interactive Maps** - Visual location representation using Leaflet.js with custom markers and popups
- **Smart Search Interface** - Quick city suggestions and intuitive search functionality
- **Comprehensive Weather Metrics** - Detailed analytics including feels-like temperature and wind direction
- **Modern User Interface** - Clean design with smooth animations and gradient styling

---

## Project Architecture

```
User Interface (HTML/CSS)
        ↓
Event Handlers (JavaScript)
        ↓
Input Validation & Processing
        ↓
Weather Data Generation/API Call
        ↓
DOM Manipulation & Rendering
        ↓
Interactive Map Display (Leaflet.js)
```

### Data flow

1. User enters city name in search input field
2. JavaScript event handler captures and validates input
3. Weather data is processed (API call or generated data)
4. Application updates DOM with weather information
5. Interactive map renders with location marker
6. Results display in structured, readable cards

---

## Usage Guide

### Searching for Weather

1. Enter a city name in the search input field &  Press Enter or click the Search button
3. View detailed weather information displayed in organized cards
4. Scroll to see the interactive map showing the city location

### Quick City Selection

- Click on any pre-defined city tag for instant results
- City tags include popular locations from India and around the world
- Tags provide one-click access to frequently searched cities

### Viewing Location Details

- Interactive map displays automatically after search
- Zoom controls allow detailed geographical exploration
- Custom markers indicate the searched city location
